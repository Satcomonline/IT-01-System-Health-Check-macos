# IT-01 System Health Check (MacOs)
A macOS system health check script that collects CPU, memory, disk, network, and basic system information into a single report. Built to simulate real-world IT/Desktop Support troubleshooting.

## What this demonstrates
- Practical desktop support troubleshooting
- MacOs system commands and diagnostics
- Basic scripting and automation
- Clear documentation

## How it works
- Run a series of system checks
- Saves results to a timestamped report on the Desktop/location of choice
- Helps diagnose performance or connectivity issues

## How to Run
chmod +x health-check.sh

./health-check.sh
